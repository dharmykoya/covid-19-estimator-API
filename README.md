## Installation

1. Ensure you have Node.js and npm installed

2. Clone this repo

```bash
$ git clone https://github.com/dharmykoya/Shibi-backend-api.git
```

3. Install Dependencies

```bash
npm install
```

4. Start server

```bash
npm run start
```
